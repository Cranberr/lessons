Here is recap of some of the big ideas:

1) The World Wide Web commonly known as WWW, here "The Web":

The Web is composed largely of HTML documents. HTML stands for HyperText Markup Language and is the language the browsers use to interpret web pages such as Udacity.

The components of The Web include:

-The Client, i.e: your computer and browser

-The Internet

-Servers.

The Client and servers interact through something called the Hypertext Transfer Protocol (HTTP)

2) Basic HTML Elements:

By using HTML elements we tell a web browser how to display content on a web page.
Most HTML elements consist of an opening and a closing tag with some content between them.

Here are some examples of very basic tags and what fun we can have with them:

- using opening tag < b >(delete spaces) before any text and closing tag < /b >(delete spaces) after this text you will make this text go completly bold

- using opening tag < em >(delete spaces) before any text and closing tag < /em >(delete spaces) after this text you will make this text go completly italic

- by analogy, opening tag < u > and closing tag < /u > will make your text nicely underlined.

Now we have tricky part:

3) HTML Attributes:

Some elements accept additional values called attributes in their opening tags. Attributes give additional information to the browser.

For example, the anchor tag takes an attribute called href that defines a URL to link to:

Udacity Creates a hyperlink to the href attribute value and names it Udacity

Displays the image hosted at the src attribute value "url". If the image cannot be displayed, then the alt attribute value "text" is shown.
